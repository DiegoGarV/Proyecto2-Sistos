# Proyecto

## Instalaciones necesarias

Primero, instala las siguientes dependencias de Python y Node.js:

```bash
pip install python-multipart
pip install fastapi uvicorn
pip install starlette
```

## Backend
Navega al backend
```cd backend```

Inicia el backend
```python -m uvicorn main:app --reload```

## Frontend
Navega al frontend
```cd frontend```

Inicia el frontend
```npm run dev```
